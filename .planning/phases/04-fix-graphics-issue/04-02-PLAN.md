**Part B: Simple drawSprite helper**

Since sprites are now at EXACT target dimensions, the helper is trivial:

```javascript
function drawSprite(img, x, y, w, h) {
  if (!img || !img.complete || img.naturalWidth === 0) return;
  ctx.imageSmoothingEnabled = false;
  ctx.drawImage(img, Math.floor(x), Math.floor(y), w, h);
}
```

Note: x,y is TOP-LEFT corner, w,h is the hitbox dimensions. No centering or
bottom-anchor math needed — the caller provides the exact position.
