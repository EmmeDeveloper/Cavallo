**Part B: Sprite Drawing Helper**

Add a universal sprite drawing function that handles bottom-anchor positioning and pixel art crispness:

```javascript
function drawSprite(img, feetX, feetY, scale) {
  if (!img || !img.complete || img.naturalWidth === 0) return;
  ctx.imageSmoothingEnabled = false;
  const w = img.width * scale;
  const h = img.height * scale;
  ctx.drawImage(img, Math.floor(feetX - w / 2), Math.floor(feetY - h), w, h);
}
```
