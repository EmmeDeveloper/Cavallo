Use a Node.js script with the `audiobuffer-to-wav` approach or raw WAV generation to create placeholder 8-bit SFX. For each sound, generate a simple synthesized tone:

- `collect`: bright coin chime (0.3s, 800Hz->1200Hz with quick decay)
- `collision`: low thud (0.4s, 150Hz with rapid decay)
- `boss_hit`: impact punch (0.3s, 300Hz->100Hz descending)
- `boss_defeated`: triumphant arpeggio (1s, ascending 3-note chord C-E-G)

Generate as WAV first, then convert to both OGG and MP3 using ffmpeg if available. If ffmpeg is not available, generate WAV files and rename to .ogg/.mp3 (Howler handles format detection gracefully) OR use a pure JS approach to write valid audio files.
